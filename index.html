return resultado; ----- Opcionalmente, se puede utilizar el operador 'return' para devolver un valor

¿Hasta cuantos argumentos puedes declarar en una función?

En JavaScript, no hay un límite específico en cuanto al número de
argumentos que puedes declarar en una función. Puedes declarar
tantos argumentos como necesites. Sin embargo, es recomendable
mantener la cantidad de argumentos bajo control para mejorar la
legibilidad y mantenibilidad del código. Si necesitas pasar muchos
argumentos, considera agruparlos en un objeto o array para simplificar
la estructura de la función. Recuerda que el objetivo es tener un código
claro y fácil de entender.
